# Dynamic Quiz Application

## Objective
Build a quiz application that dynamically loads questions and tracks user scores.

## Requirements

### 1. Question Management
- Store quiz questions and options in a JavaScript object  
  **OR**
- Load them from an external JSON file

### 2. User Interaction
- Use event listeners to:
  - Capture user selections
  - Navigate through quiz questions dynamically

### 3. Scoring System
- Calculate the user's score based on correct answers
- Track progress throughout the quiz

### 4. Result Display
- Display the final score at the end of the quiz
- Provide:
  - Feedback based on performance
  - Optional explanations for correct answers
